• help支援的瀏覽器為何？

    以下的瀏覽器已經測試，並確定可以使用：Google Chrome 35+、Firefox 31+。
    其他新款瀏覽器應該都可以使用。Internet Explorer 可能無法進行人機驗證。我們亦不保證在 Microsoft Edge 上能正常運作。

  • help如何登入？

    登入步驟如下：
    1. 輸入使用者資訊。請選取班別與輸入學號。
    2. 輸入密碼。注意密碼的大小寫必須完全準確。
    3. 進行人機驗證。按下 [我不是自動程式] 方塊以確認並非自動程式正在使用此系統。如有需要，請回答驗證問題。

  • help如何報名？

    在報名頁面上選取您欲報名的項目。
    個人項目方面，您需要由第一個項目順序填寫到第三個項目（此僅為方便系統紀錄，實際項目次序不限）。選取項目不可重複。當取消選取某個項目時，其他項目會依序移前。
    接力項目方面，請勾選您報名的項目。請務必在填寫此欄位前查閱報名須知。
    當報名時段過後，您仍然能在報名頁面檢視您已報名的項目，但您不能夠再更改項目。

  • help發生了錯誤該怎麼辦？

    請聯絡系統開發者以修正錯誤或取得解決方法。提交錯誤報告時，請儘量詳盡描述，並提供錯誤截圖，以便開發人員研究問題。
    聯絡方式：請到 GitHub 建立新的問題 (Issue) 以便開發人員集中處理。
